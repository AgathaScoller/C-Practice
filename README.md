Repository dedicated to **studying and practicing C programming**.  
Focus is on mastering procedural programming, memory management, data structures, and algorithmic logic.

---

## STRUCTURE

- **Basics/** – syntax, variables, data types, operators, input/output  
- **ControlFlow/** – conditionals, loops, and branching  
- **Functions/** – modular programming and reusable code  
- **DataStructures/** – arrays, structs, pointers, and linked lists  
- **Projects/** – small programs and exercises to apply concepts  
- **Utils/** – helper functions and templates  

---

## OBJECTIVES

- Strengthen understanding of **procedural programming**  
- Practice memory management, pointers, and data structures  
- Build a foundation for **systems programming, embedded applications, and algorithmic problem-solving**  

---

## ENVIRONMENT

- **Language:** C  
- **Compiler / Tooling:** GCC, Clang, or any standard C compiler  
- **Editor / IDE:** VS Code, Code::Blocks, or any IDE of choice  
- **Platform:** Cross-platform (Windows, Linux, macOS)  

---

## CODE STYLE

- Code emphasizes **clarity, modularity, and efficiency**  
- Comments are used to explain complex logic or memory handling  
- Functions and variables follow consistent naming conventions  
- Programs follow standard C programming best practices
